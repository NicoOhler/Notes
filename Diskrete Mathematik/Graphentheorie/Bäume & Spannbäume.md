# Bäume & Spannbäume
### Bäume
+ Graph ohne Kreis heißt Wald
+ Grap heißt Baum, wenn
	+ zusammenhängend
	+ ohne Kreis <==> $|E|=|v|-1$
+ für je zwei Knoten, gibt es genau einen Pfad

### Path Finder
+ Input: Baum T, Knoten x,y
+ Output: kürzeste Pfad von x nach y
+ Algorithmus
	+ wähle Knoten $x_0$ aus $V/\{x,y\}$ als Wurzel
