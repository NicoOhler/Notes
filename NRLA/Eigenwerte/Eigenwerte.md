# Eigenwerte und Eigenvektoren
 + F: V --> V
	 + lineare Selbstabbildung
	 + $Av=λv$
		 + v - Eigenvektor
		 + λ - Eigenwert
		 + A quadratisch
	 + dim(V) endlich ==> EW/EV Problem für Matrizen

### Eigenwert/Eigenvektor-Problem
 + $Av=λv=λIv$
	 +  $(A-λI)v=0$
		 +  homogenes lineares Gleichungssystem
		 +  nichttriviale Lösung, wenn $det(A-λI)=0$
