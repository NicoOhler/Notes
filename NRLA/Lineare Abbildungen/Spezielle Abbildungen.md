# Spezielle Abbildungen
+ Zoom-Abbildung
	+ Z<sub>λ</sub>: $ℝ^2$ --> $ℝ^2$
	+ (x, y) --> λ(x, y)
	+ Stauchung für λ > 1
	+ Streckung für 0 < λ < 1
+ Skalierung
	+ S<sub>λ,μ</sub>: $ℝ^2$ --> $ℝ^2$
	+ (x, y) --> (λx, μy)
+ Spiegeln an Koordinatenachsen
	+ Sp<sub>x</sub>: $ℝ^2$ --> $ℝ^2$
	+ (x, y) --> (x, -y)
+ Scherung
	+ Sch<sub>x</sub>(α): $ℝ^2$ --> $ℝ^2$
	+ (x, y) --> (x + tan(y), y)
