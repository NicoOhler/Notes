# Zufallsvariable
### Definition
+ Zufallsvariable ZV
	+ $X:Ω→Ω'$
		+ oft $Ω'=ℕ/ℝ$
	+ $ω→X(ω)$
	+ weist jedem Elementarereignis einen Wert zu
	+ erzeugt Ereignisse - Urbild von Ereignissen
		+ ![[Pasted image 20221013061951.png]]
+ diskret, wenn X nur abzählbar viele Werte annimmt
+ stetig, wenn nicht abzählbar
