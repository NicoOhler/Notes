### Definition
+ Urne enthält N Kugeln mit $L ≥ 2$ Farben
+ $N_i$ Kugeln mit Farben $L_i$
+ $N=N_1+...+N_L$
+ n mal Ziehen ohne Zurücklegen

### Eigenschaften
+ $Ω=\{\{j_1,...,j_n\}⊆\{1,...,N\}\}$
	+ $|Ω|=\binom{N}{n}$
+ $A=P(Ω)$
+ $P(\{j_1,...,j_n\})=\frac{1}{\binom{N}{n}}$
	+ $P(A)=\frac{|A|}{\binom{N}{n}}$
