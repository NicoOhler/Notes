# Logische Verknüpfungen
####  Konjunktion:
+ Mögliche Schreibweisen:
	+ ∧
	+ und/and
	+ &
	+ &&
+ Wahrheitstabelle

	A | B | A∧B 
	--- | --- | ---
	True | True | True 
	True | False | False 
	False | True | False 
	False | False | False 
	
#### Disjunktion: 
+ Mögliche Schreibweisen:
	+ ∨
	+ oder/or
	+ ||
+ Wahrheitstabelle

	A | B | A∨B 
	--- | --- | ---
	True | True | True 
	True | False | True 
	False | True | True 
	False | False | False 
	
	#### Negation:
	+ Mögliche Schreibweisen:
		+ ¬
		+ nicht/not
		+ !
+ Wahrheitstabelle

	A |  ¬A
	--- | ---
	True | False
	False | True
	
#### Implikation/Subjunktion: 
+ Wenn A, dann B: A→B
+ Mögliche Schreibweisen:
	+ →
	+ impliziert/implicates
+ Wahrheitstabelle

	A | B | A→B 
	--- | --- | ---
	True | True | True 
	True | False | False 
	False | True | True 
	False | False | True 
	
#### Bijunktion/Äquivalenz?: 
+ wahr, wenn beide Variablen **denselben** Wahrheitswert haben.
+ Mögliche Schreibweisen:
	+ ↔
	+ gleich/equal to/is
	+ ==
	
+ Wahrheitstabelle

	A | B | A↔B 
	--- | --- | ---
	True | True | True 
	True | False | False 
	False | True | False 
	False | False | True
	
#### Tautologie: 
+ **immer** wahr
+ wird benutzt für Aussagen welche immer wahr sind
+ Mögliche Schreibweisen:
	+ ⇔ (Äquivalenz, welche immer wahr ist)
	+ ⇒ (Implikation, welche immer wahr ist)
+ Beispiele: ∨ ∧
	1. A∨B ⇔ B∨A
	1. A∧B ⇔ B∧A
