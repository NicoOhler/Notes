### Definition
+ proven and established solution for a reoccurring problem
	+ idea, scheme or template
	+ general and unspecific, suitable for a class of problems

### Purpose
+ efficient and fast since ideas are reused
+ reduces misunderstandings with established terminology
+ easier knowledge transfer
+ increases usefulness of specific ideas by generalizing to different problems

### Format
+ pattern name
+ context
	+ situation in which a problem occurs
+ problem
+ forces
	+ requirements, constraints and challenges
	+ motivating factor
	+ why is the problem difficult in this context
+ solution
	+ generic description of the implementation
	+ describes components and how they interact
	+ structures, behaviors, actionable steps, ...
+ consequences
	+ benefits and draw backs
	+ how are the forces resolved
	+ liabilities, limitations and tradeoffs
+ known uses
	+ real world examples
