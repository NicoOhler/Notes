### Overview
+ not only SQL
+ [[SQL]] alternative
+ uses [[Document Stores]] to store data as documents within collections as [[Key-Value Pairs]]

### Benefits
+ non-relational
	+ key value pairs
	+ graphs
	+ time series
+ schema-free
	+ flexible
	+ implicit
+ impedance mismatch
+ scalable
	+ horizontal partitioning (sharding)
	+ scaling
