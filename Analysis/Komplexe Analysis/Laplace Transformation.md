+ f: $[a,\infty)$-->ℝ
+ Laplace Transformation 
	+ ![[Pasted image 20220608124719.png]]
	+ $Lf(s)$ f existiert, wenn $∃A,C∈ℝ ∀t∈ℝ^+:|f(t)|≤Ce^{At}$

	+ $Lf(s)$ ist holomorphe Funktion für $Re(s)>c_0(f)$
