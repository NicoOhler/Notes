# Differentialgleichungen
### Definition
+ Sei F: $ℝ^{n+2}$-->ℝ
	+ $F(x,y,y',...,y^{(n)})$ ist eine Differentialgleichung
	+ x unabhängig
	+ y abhängig
	+ Beispiel:
		+ $xy'+y^2+y''=0$
		+ DGL 2. Ordnung
		+ gesucht y=y(x)
