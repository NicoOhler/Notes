# Getrennte Variablen
### Typ 1: getrennte Variablen
+ x und y voneinander trennbar
	+ gegeben: $y'=\frac{f(x)}{g(y)}=\frac{dy}{dx}$
	+ nach Umformung: $g(y)dy=f(x)dx$
	+ nach Integration: $G(y)=F(x)+C$
