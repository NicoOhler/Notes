# Elementare Funktionen
### Exponentialfunktion
+ exp: ℝ->ℝ
+ x->$\sum_{n=0}^∞ \frac{x_n}{n!}$
+ konvergiert für alle x∈ℝ
	+ sogar für x∈ℂ
+ stetig
+ streng monoton wachsend
+ injektiv
+ Eigenschaften
	+ $exp(x+y) = exp(x*y)$
	+ $exp(0) = 1$
	+ $∀x∈ℝ(ℂ): exp(0)≠0$
	+ $∀x∈ℝ: exp(x)>0$
	+ $exp(-x) = \frac{1}{exp(x)}$
	+ $∀x>0: exp(x)>1$
		+ $exp(x)>1+x$ für $x>0$
	+ $exp(1)=e$ ==> $exp(n)=e^n$
		+ $exp(-1)=\frac{1}{e}$ ==> $exp(-n)=\frac{1}{e^n}$
